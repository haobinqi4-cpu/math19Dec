## Quick Start Guide

### For Business Analytics
Start with: **pandas** → **NumPy** → **Matplotlib** → **Jupyter** → **scikit-learn**

### For Fintech Analytics
Start with: **pandas** → **yfinance** → **Matplotlib/Plotly** → **Backtrader** → **TA-Lib** → **QuantLib**

## Installation

Most libraries can be installed via pip:
```bash
pip install pandas numpy matplotlib scikit-learn jupyter yfinance plotly backtrader
```

Some require additional setup:
- **TA-Lib**: Requires C library installation first
- **QuantLib**: Requires QuantLib-Python package
