# Walmart Sales Data Visualizations

This directory contains visualizations of the Walmart sales dataset in HTML format.

## Contents

- `walmart_visualization_report.html` - Interactive HTML report with visualizations and descriptions

## Visualizations Included

1. Total Weekly Sales Over Time
2. Average Weekly Sales by Store
3. Sales Distribution: Holiday vs Non-Holiday Weeks
4. Correlation Matrix of Numerical Variables
5. Distribution of Weekly Sales

The HTML report contains embedded images and textual descriptions for each visualization.
